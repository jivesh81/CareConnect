## ğŸ“¦ Installation

### Prerequisites

```bash
node >= 16.x
npm >= 8.x
PostgreSQL >= 13.x
MongoDB >= 5.x
Redis >= 6.x
```

### Clone Repository

```bash
git clone https://github.com/jivesh81/careconnect.git
cd careconnect
```

### Backend Setup

```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables in .env
npm run migrate
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
cp .env.example .env
# Configure API endpoint in .env
npm start
```

### Docker Setup (Recommended)

